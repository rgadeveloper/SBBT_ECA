Estructura del archivo a cargar

Esta funcionalidad permite al usuario realizar la carga masiva en el simbal de los amarres/desamarres de trafos a macros virtuales.

El archivo debe tener la siguiente estructura:

IDMACROVIRTUAL;IDCOMERCIALTRAFO;PERIODO;TIPO

El campo ‘TIPO’ se refiere al tipo de accion a realizar:

  • Crear relación. TIPO = A
  • Terminar relación. TIPO = D

Ejemplo:

MVG061212;T161110455;201209;A

MVG061212;T161070210;201209;D
